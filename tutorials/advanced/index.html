Advanced Techniques

Level up your skills with advanced terminal usage, scripting, and privacy tools. Prerequisites: Complete the basics tutorial first. Note; these terminals are used for UNIX and attached systems, not Windows based.

Advanced Topic 1: Shell Scripting

Automate repetitive tasks by writing shell scripts.

Your First Script

#!/bin/bash # backup.sh - Simple backup script echo "Starting backup..." tar -czf backup-$(date +%Y%m%d).tar.gz ~/Documents echo "Backup complete!"

Make it executable: chmod +x backup.sh

Run it: ./backup.sh

Advanced Topic 2: Network Tools

Monitor and secure your network connections.

Useful Commands

$ ping google.com # Test connectivity $ netstat -tuln # View open ports $ ssh user@host # Secure remote login $ scp file.txt user@host:~ # Secure copy

Always use these tools ethically and legally. Never scan networks you don't own or have permission to test.

Advanced Topic 3: Privacy Tools

Tor Browser (Command Line)

# Install Tor $ sudo apt install tor # Start Tor service $ sudo systemctl start tor # Configure proxy: SOCKS5 localhost:9050

GPG Encryption

$ gpg --gen-key # Generate key pair $ gpg --encrypt file.txt # Encrypt a file $ gpg --decrypt file.txt.gpg # Decrypt a file

Advanced Topic 4: System Monitoring

$ top # Real-time process monitor $ htop # Enhanced process viewer $ df -h # Disk space usage $ du -sh * # Directory sizes $ ps aux # List all processes

Use q to quit top/htop viewers.
