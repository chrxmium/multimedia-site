Terminal Basics

Master the fundamentals of the command line. This tutorial covers essential commands and concepts for beginners. Note; these terminals are used for UNIX and attached systems, not Windows based.

Lesson 1: Understanding the Terminal

The terminal (or command line) is a text-based interface to your computer. Instead of clicking icons, you type commands.

Why use the terminal?

  • More powerful and flexible than GUI tools
  • Required for many privacy and security tools
  • Faster once you learn the basics
  • Enables automation through scripting

Lesson 2: Essential Commands

Navigation

$ pwd # Print working directory $ ls # List files $ cd Documents # Change directory $ cd .. # Go up one level $ cd ~ # Go to home directory

File Operations

$ touch file.txt # Create empty file $ mkdir new_folder # Create directory $ cp file.txt backup.txt # Copy file $ mv file.txt newname.txt # Move/rename $ rm file.txt # Remove file

Lesson 3: Viewing and Editing

$ cat file.txt # Display file contents $ less file.txt # View file (paginated) $ head file.txt # Show first 10 lines $ tail file.txt # Show last 10 lines $ nano file.txt # Edit with nano editor

Tip: Use Tab for auto-completion and ↑/↓ arrows for command history!
